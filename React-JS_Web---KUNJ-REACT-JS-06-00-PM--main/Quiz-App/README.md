## ğŸ”§ Customization

### Adding New Questions
Edit the `quizData` object in `App.jsx`:

```javascript
const quizData = {
  easy: [
    {
      question: "Your question here?",
      options: ["Option A", "Option B", "Option C", "Option D"],
      correct: 2, // Index of correct answer (0-3)
      explanation: "Detailed explanation of the answer"
    }
  ]
}
```

### Modifying Difficulty Settings
Change time limits in the `getTimeForDifficulty` function:

```javascript
const getTimeForDifficulty = (diff) => {
  switch (diff) {
    case 'easy': return 45;    // seconds
    case 'medium': return 30;
    case 'hard': return 20;
    default: return 30;
  }
};
```

### Styling Customization
Modify CSS variables in `index.css` for consistent theming:

```css
:root {
  --primary-color: #667eea;
  --secondary-color: #764ba2;
  --success-color: #4CAF50;
  --error-color: #f44336;
  --warning-color: #FF9800;
}
```
